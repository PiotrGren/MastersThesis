# Master Thesis Project

## Subject of the thesis: AI-driven log analysis for user behavior classification

### Abstract

This repository contains files from a project completed as part of a master's thesis. The repository consists of two main subfolders:

1. **FinanceApp** - scripts for the financial exchange application and the load application. The application is designed to simulate financial transactions, simulate user traffic using the load application, and monitor and log user behavior.
2. **Analysis** - AI model scripts and user behavior data analysis. The models will be trained to classify users based on their behavior (logs collected from application simulations).
